# CS-370

During my time in this course I have learned valuable lessons about AI and machine learning, ethical and social implications of these concepts, and how to apply them to tasks effectivly.
In this repository I have the Jupyter notebook file containing the code I used for my submission for project 2. The goal of the program was to start the free agent off in an uncharted maze and using a Deep Q-Learning algorithm the pirate would get itself closer and closer to the end goal, the treasure, in the maze over time. I applied a reward structure to guide the pirate in the right direction towards teh treasure and away from the walls and invalid move, the pirate agent stores all of these experiences in its memory buffer. Using this stored experience the agent is able to, over enough iterations, make the best possible move each time to make it to the treasure 100% of the time. Upon doing so the program ends and a final simulation is ran in which the pirate will reach the treasure.
